Arcra

an Arch based distro inspired by the Nobara Project.

This website is a work in progress.

Download Arcra

You can download Arcra by subscribing to this RSS feed. It will provide a torrent file now and each time a new ISO file is available, as well as the relevant checksum files; (md5 and sha512) for you to verify the ISO hasn't been modified in any way during transit.

Latest Release:  arcra-gnome-2023.04.30-x86_64.iso

The username and password for the ISO are:  liveuser and root, respectively.

Arcra is only available for 64-bit architectures, as it is a gaming/content creation orientated linux distro, just like Nobara.
